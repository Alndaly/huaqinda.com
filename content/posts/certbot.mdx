---
title: 基于certbot自动生成ssh证书
---

基于 Nginx 方式最为方便

## **1. 安装 Certbot**

ubuntu

```bash
sudo apt update
sudo apt install certbot python3-certbot-nginx  # 如果用Nginx
sudo apt install certbot python3-certbot-apache  # 如果用Apache
```

macos

```bash
brew install certbot
```

## 2. 获取 SSL 证书

```bash
sudo certbot --nginx
```

这个脚本可以一键配置

你只需要跟着提示走一遍就好

## 3.自动续期

默认 certbot 会开启一个定时服务进行自动续期，不需要你再配置了

手动续期命令

```bash
sudo certbot renew --dry-run
```
